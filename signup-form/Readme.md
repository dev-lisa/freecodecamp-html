# Sign-Up Form UI

A clean and responsive sign-up (registration) form built using HTML and CSS.

## ✨ Features

* Full name input field
* Email input field
* Password field
* Confirm password field
* Simple and clean user interface

## 🛠️ Built With

* HTML5
* CSS3

## 📌 Purpose

This project was created as part of my web development learning journey to practice building user interfaces.

## 🚀 Future Improvements

* Add JavaScript for form validation
* Connect to a backend for real user authentication
