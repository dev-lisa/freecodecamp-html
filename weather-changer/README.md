Weather Changer App 🌦️

A simple interactive weather application built using HTML, CSS, and JavaScript. The app provides recommendations and suggestions based on different weather conditions entered by the user.



Features

Displays weather-based recommendations

Suggests items such as umbrellas, sunglasses, or warm clothing

Interactive user input

Responsive and beginner-friendly design



Technologies Used

HTML5

CSS3

JavaScript


How to Run the Project

1. Clone the repository



https://github.com/dev-lisa/freecodecamp-html.git

2. Open the project folder in VS Code


3. Open index.html in your browser
or use the Live Server extension
